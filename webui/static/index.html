- -

格式说明

sub2api JSON

通用交换格式，顶层含 type / version / exported_at / proxies / accounts，账号按 platform 区分 grok / openai。

Grok CPA 包

CLIProxyAPI 的 xai-*.json，字段含 access_token / refresh_token / expired / token_endpoint 等，与本项目 cpa_auths 产物一致。

OpenAI CODEX 包

codex-*.json，含 access_token / refresh_token 或 session_token、account_id、plan_type 等。

识别规则

auto 模式：输入为 sub2api 则按 platform 拆出 CPA / CODEX 两套 zip；输入为原生格式（单 json / 目录打包 zip / json 列表）则合并转为 sub2api。
